Multiple values wizard

If a column contains regularly appearing values of a different type that have corresponding values somewhere else in the data - this wizard aims to reorganise them into their own columns with those corresponding values.

Example An example could be a column named "Energy type" that contained the same repeating 3 values "Electricity, Water, Gas" with another column named "Readings" that contained their corresponding energy reading values. After running this wizard, the "Energy type" column would be transformed into three new columns - "Electricity", "Water" and "Gas", that each contain their corresponding values from the "Readings" column.

Select the column containing multiple types:

Start Select To select multiple column headings, hold down CTRL (Windows) or Apple (Mac). You can also drag your mouse across multiple column headers while holding down the relevant key.

Click anywhere in the table to unselect the columns.

Select the column containing their corresponding values:

Start Select To select multiple column headings, hold down CTRL (Windows) or Apple (Mac). You can also drag your mouse across multiple column headers while holding down the relevant key.

Click anywhere in the table to unselect the columns.

Select the columns that contain values that do not correspond to the multiple types (i.e. a "Totals" column):

Start Select To select multiple column headings, hold down CTRL (Windows) or Apple (Mac). You can also drag your mouse across multiple column headers while holding down the relevant key.

Click anywhere in the table to unselect the columns.
