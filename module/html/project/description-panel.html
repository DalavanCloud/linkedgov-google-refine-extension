Labels and descriptions

To make this data searchable and useful for others to use on the web, it's important that the columns have meaningful names and small one-line descriptions. It is not expected that there is an appropriate description for the rows and columns, however please try to provide them with a decent label.

Key:

  • - Needs checking
  • - Good label, but no description
  • - Great!
