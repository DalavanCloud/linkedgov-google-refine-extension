Blank values wizard

The blank values wizard is used to blank out any "missing" values in the data - which is a better way of storing blank values than using a particular value.

"Blank" values exist in many different forms across datasets (e.g. "NULL", "missing", "-"), so it's important for us to recognise that these values are indeed blanks and not potentially correct values that should be processed later on.

  1. Try to find your dataset's own "missing" or "blank" value
  2. Enter that value in the text box below
  3. Click "Update"

You should then see your "missing" values be wiped clean from the table.
