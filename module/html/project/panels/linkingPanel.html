The linking panel will attempt to identify "things" in the data that are already defined somewhere on the web. By linking them to existing definitions, it helps machines realise that the data is describing the same "things" as someone else has.

The list of things to link to is very limited for the time being - so don't worry if there's nothing to link!

Click on the column names to confirm they have been guessed correctly - otherwise have a go at linking the data yourself.
