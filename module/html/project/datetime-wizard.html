Date and time wizard

This wizard will try to understand and format any dates or times you have in your data.

You might find dates in a single column or perhaps split across multiple columns. To begin, select any columns that contain a complete date, then repeat this process if there are any other dates:

Start Select To select multiple column headings, hold down CTRL (Windows) or Apple (Mac). You can also drag your mouse across multiple column headers while holding down the relevant key.
